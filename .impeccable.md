## Design Context

### Users
Developers and crypto-native power users who delegate USDC spending to autonomous AI agents on Base Sepolia. They need a clear dashboard to set guardrails, monitor agent activity, and file disputes when an agent overspends. Used primarily on desktop during focused work sessions.

### Brand Personality
Vigilant, precise, trustworthy. IntentGuard is the vault door between an AI agent and your money. It should feel like a high-security control room — authoritative without being intimidating, data-dense without being cluttered.

### Aesthetic Direction
**Industrial precision on dark canvas.** Deep blue-black backgrounds with warm gold accents — the visual language of vault doors, security badges, and control rooms. Not the cyan-on-dark "AI dashboard" cliche. Gold communicates trust and value protection.

- Theme: Dark (users are developers monitoring agent activity in focused sessions)
- Display font: Bricolage Grotesque (geometric with optical quirks, authoritative)
- Body font: Red Hat Text (engineered, precise, excellent readability at small sizes)
- Palette: Deep navy base, warm gold accent, semantic status colors (green/red/amber)
- Layout: Left-aligned, asymmetric, data-dense. No card-in-card nesting. No uniform grids.

### Design Principles
1. **Show the math** — Every number (amounts, caps, stakes) should be immediately verifiable. No hiding data behind clicks.
2. **Status at a glance** — Receipt/challenge states use color + icon + label redundantly so nothing is ambiguous.
3. **One action per context** — Each screen has one primary action. Don't compete for attention.
4. **Earned trust** — The interface should make the user feel in control, not impressed by decoration.
5. **Hackathon honest** — Ship clean, working code. No fake polish over broken flows.
